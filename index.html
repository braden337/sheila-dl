youtube-dl usage for Sheila

  1. Download the latest version of youtube-dl for Windows here. There is a link near the top of the page with the text Windows exe.
  2. Press Windows Key + R to open the Run window.
  3. Type powershell and press enter (if that doesn't work try cmd).
  4. Type cd path-to-folder to change to the directory that contains youtube-dl.exe eg: cd "F:\Demo Folder".
  5. To display the video/audio formats available for download, run the command: 
    ./youtube-dl https://www.youtube.com/watch?v=G8KpPw303PY -F
  6. It should display something like this:

  7. To download a particular format, for example format code 36, the .3gp small video file, instead of -F, use the lowercase -f flag like -f 36 after the YouTube® video URL. It would look like this: 
    ./youtube-dl https://www.youtube.com/watch?v=G8KpPw303PY -f 36
